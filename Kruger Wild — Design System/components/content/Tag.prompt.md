Soft pill for habitats, categories, trail difficulty, and filter chips.

```jsx
<Tag tone="green" icon={<i className="ph ph-tree" />}>Riverine forest</Tag>
<Tag tone="ochre" interactive selected>Big Five</Tag>
```

Tones: `neutral | green | ochre | clay | teal`. Use `interactive`+`selected` for filter rows. Sizes `sm | md`.
