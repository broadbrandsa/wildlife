Pill-shaped primary action; use for any call-to-action, form submit, or nav CTA across Kruger Wild surfaces.

```jsx
<Button variant="primary" iconRight={<i className="ph ph-arrow-right" />}>
  Plan your visit
</Button>
```

Variants: `primary` (filled bushveld green — default CTA), `accent` (savanna gold — high-emphasis booking/donate), `secondary` (green outline — paired secondary action), `ghost` (quiet inline action), `ondark` (sand fill, for use over green/photographic sections). Sizes `sm | md | lg`. Pass `iconLeft` / `iconRight` a Phosphor `<i className="ph ph-…" />`. Set `fullWidth` inside narrow cards or mobile sheets.
